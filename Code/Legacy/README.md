# Legacy Reference Code

This folder stores original Dragon Alpha Carbon-era source snapshots for parity reference only.
These are Dragon Alpha legacy sources archived in this repository, not external runtime/source snapshots.

- `Source/`: original gameplay/runtime implementation sources.
- `Headers/`: original gameplay/runtime headers.
- `JNP Libraries/`: original platform utility layer used by the classic codebase.

Modern runtime code lives in `Code/` and is the only code compiled by `Dragon Alpha.xcodeproj`.
