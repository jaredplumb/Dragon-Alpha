# Images

Designer-facing image assets.

- `Source/DragonArcadia/art`: canonical editable masters (`cdr`, `psd`, `cpt`, etc).
- `Legacy/Resources`: historical PNG set from the legacy project.
- `../Legacy/Art/Additional`: canonical legacy item/monster JPG source library.
- `RuntimeOverrides`: curated PNG overrides promoted to `Resources/Images` by local asset-import workflow.
- `Catalog/Runtime`: categorized links to `Resources/Images/*.png`.
- `Catalog/Source`: categorized links to best-known source assets for each runtime image.
