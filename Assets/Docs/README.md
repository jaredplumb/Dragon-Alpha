# Generated Asset Docs

Canonical generated manifests for the active Dragon Alpha asset workspace.

- `RuntimeAssetSourceMap.latest.tsv`: runtime `Resources/` to `Assets/` provenance map.
- `AssetSourceOfTruth.latest.tsv`: current media source-of-truth selection across `Assets/Images`, `Assets/Sounds`, `Assets/Fonts`, and `Assets/Music`.
- `CdrPsd-Decisions.latest.tsv`: current CDR/PSD pair retention recommendations.
- `Midi-Source-Manifest.tsv`: staged original MIDI name-to-source mapping.

Historical one-off recovery notes were moved to:
- `Assets/Archive/Deprecated/RecoveryNotes/`

Older timestamped generated snapshots are intentionally omitted from this public repository.
