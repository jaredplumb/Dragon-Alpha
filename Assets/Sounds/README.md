# Sounds

Designer-facing sound assets.

- `Source/DragonArcadia/sounds`: canonical source WAV files from DragonArcadia import.
- `Legacy/Resources`: historical runtime WAV files.
- `Legacy/PowerPlug`: legacy AIFF source sounds.
- `RuntimeOverrides`: curated WAV overrides promoted to `Resources/Sounds` by local asset-import workflow.
- `Catalog/Runtime`: categorized links to `Resources/Sounds/*.wav`.
- `Catalog/Source`: categorized links to best-known source assets for each runtime sound.
