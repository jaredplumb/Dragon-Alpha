# Archive

Minimal archive surface kept in the public-facing repo.

- `DragonAlpha.dmg`: preserved original open-source release archive (see `ArchiveManifest.md`).
- `Deprecated/RecoveryScratch/`: canonical recovery-scratch path retained for local validator compatibility.
- `Deprecated/RecoveryNotes/`: historical one-off recovery/import notes kept for provenance.

Large historical archive snapshots and extraction debris are intentionally omitted from this public repository.
