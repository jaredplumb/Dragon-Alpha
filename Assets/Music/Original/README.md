# Original MIDI Sources

- `MIDI/`: deduplicated symlink stage of original/legacy MIDI material.
- staged filenames keep source context where possible (`Caves__battle.mid`, `Core__map.mid`, `Carved__carve_0009.mid`) instead of generic `Music__...` aliases.
- Generated by local MIDI staging workflow.
- Source mapping is tracked in `Assets/Docs/Midi-Source-Manifest.tsv`.
