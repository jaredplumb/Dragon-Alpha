# Dragon Alpha Design

This folder is the human-facing product/release workspace.

## Core Docs
- `01.Product-Guide.md`: durable product direction and milestone intent.
- `02.Gameplay-Guide.md`: durable gameplay-loop and progression authority.
- `03.Visual-Guide.md`: durable visual/layout authority for release quality.
- `Backlog.md`: prioritized implementation/release work.
- `Testing.md`: canonical deterministic validation inventory (`DRAGON_TEST`).
- `Release-Readiness.md`: first-push gate, native-only signoff, and push sequence.
- `Changelog.md`: concise dated changes for active work.

## Supporting Docs
- `MidiRuntimeMapping.md`: current staged MIDI source map and runtime intent.
- `SaveSchemaMigrationPolicy.md`: save/version compatibility policy.
- Runtime tuning config now lives at `Resources/Config/RuntimeDesign.cfg`.

## Operational Note
- This folder is intended to be self-contained for public readers.
- Build/runtime operation does not require any private local automation folders.
